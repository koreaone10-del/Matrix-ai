🌐 الاتصال المباشر و CORS

كل الطلبات تُرسل مباشرة من متصفحك لخادم كل مزوّد — بدون أي خادم وسيط من هذا التطبيق. هذا يعمل فقط إذا سمح المزوّد بذلك صراحة (CORS)، وهو أمر يتحكم فيه خادم المزوّد وليس هذا التطبيق، ولا توجد طريقة من جهة المتصفح فقط لتجاوزه إن رفضه المزوّد.

المزوّدون المدرجون افتراضياً هنا يدعمون النداء المباشر من المتصفح. إن ظهر لمزوّد ما وضع "🚫 محظور CORS" في تبويب المفاتيح، فهذا يعني أن خادمه لا يسمح بذلك حالياً — الحل الوحيد الصحيح هو بروكسي شخصي تنشره أنت بنفسك (وليس بروكسي عام مشترك مثل corsproxy.io، لأن أي بروكسي عام يرى مفاتيحك وقد يسجّلها، وهذه الخدمات غالباً غير مستقرة).

📋 كود جاهز لنشر بروكسي خاص بك على Cloudflare Workers (مجاني، دقيقتان)
export default { async fetch(req) { const target = new URL(req.url).searchParams.get('url'); if (!target) return new Response('missing ?url=', {status:400}); const upstream = await fetch(target, { method: req.method, headers: req.headers, body: ['GET','HEAD'].includes(req.method) ? undefined : req.body }); const res = new Response(upstream.body, upstream); res.headers.set('Access-Control-Allow-Origin', '*'); res.headers.set('Access-Control-Allow-Headers', '*'); return res; } }

انشره على workers.dev (مجاني ضمن حساب Cloudflare الخاص بك)، ثم الصق رابطه أعلاه. هذا الرابط تحت سيطرتك أنت فقط.

ما هو هذا التطبيق: واجهة دردشة/توليد وسائط/منافسة نماذج تعمل بمفاتيحك الخاصة (BYOK) مع عدّة مزوّدين (أغلبهم بدون بطاقة، بدون تحقق هاتف)، وتبديل تلقائي عند تعطل أي مزوّد.

ما لم يُبنَ عمداً: لا يوجد تجاوز لحماية CORS من جهة المتصفح (هذا غير ممكن تقنياً أصلاً)، ولا التفاف على الحظر الجغرافي/IP، ولا الالتفاف على التوكنز/المصادقة المطلوبة من أي مزوّد. المزودون المدرجون يدعمون النداء المباشر أصلاً؛ وإن احتاج أحدهم بروكسي مستقبلاً فهو بروكسي شخصي تنشره أنت (قسم "الاتصال المباشر و CORS")، وليس خدمة عامة مشتركة.

عداد التوكنز: تقدير تقريبي (≈4 حروف/توكن للاتينية، ≈2.3 للعربية)، إلا مع Pollinations حيث يمكن مزامنة الرصيد الحقيقي بالدولار من حسابك عبر زر "مزامنة الرصيد الحقيقي" في نافذة الاستخدام.

شريط تقدّم الصور/الفيديو: محاكاة زمنية تقريبية — لا توجد نسبة تقدّم حقيقية من مزوّدي الوسائط حالياً.

رفع الملفات: الصور تُفهم مباشرة (رؤية) وتُستخدم كمرجع للتعديل/الفيديو. ملفات الكود/النص تُدرَج كنص داخل الرسالة. لا يوجد مزوّد هنا يقبل فيديو كمدخل مباشر للدردشة حالياً.

الفحص الدوري: يعمل فقط أثناء فتح التطبيق في المتصفح، وليس خدمة نظام تعمل 24/7.
